Comandos básicos Git & GitHub

Flujo esencial para trabajar con repositorios

01

Inicializar proyecto

mkdir proyecto-git
cd proyecto-git
git init
02

Primer commit

git add .
git commit -m "proyecto base"
git status
03

Crear repo en GitHub

Crear repositorio desde la web

04

Vincular con GitHub

git remote add origin URL_DEL_REPO
05

Push

git push -u origin master
06

Actualizar cambios

git add .
git commit -m "nuevo cambio"
git push origin master
